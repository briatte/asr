# README

The `basics.r` file is a script that contains code written in the R programming language. You can open it in any text editor, but I am going to ask that you open it with RStudio by doing the following:

1. Open RStudio, and take a quick look around the interface. We will be using that interface in class, every single week. I will walk you through its basics.
2. Go the Session menu, and select __Session > Set Working Directory… > Choose directory…__. Then, find the `exercise-00` folder on your hard drive, and select it.
4. Now, look around the RStudio interface: there is pane called __Files__ somewhere. Click on it, and you should see a list of files, including `basics.r`.
5. Click on the script (which is how we call a text file containing code), which will open it in its own window.

Read through the code first, and get used to the syntax colors. Comments start with a `#` hashtag, and are meant for you to read them. The rest is R code.

To execute a line of code, put your mouse anywhere on the line, and press `Ctrl-Enter`. You will see a result in the Console. A few lines do not produce any result, at least not visibly.

The code should be executed from top to bottom. Do not skip lines, or the next ones might not work, in which case you will get an error message.

Simply go through the script by reading the comments and executing the line of code below it as you go. A few things might not make sense, which is perfectly fine, but still, try to understand how the code is written (its syntax), and also try to understand what it does (its results).

Like I said, a few lines won't produce any results, and at some point, you will also get a warning (which is different from an error). Warnings are fine.

There is a very short exercise somewhere in the code. It uses data from the [U.S. General Social Survey][gss] (2000-2014). Give it a go!

[gss]: https://gss.norc.org/

And if things stop working, just try again. Using R and RStudio will take practice. This is your first practice session.
